## How to install MYSQL and PostgreSQL.
### Introduction.
MYSQL is just but an open-source database. It is a relational database system where one can manage, store and extract data that is structured. PostgreSQL is an advanced database system that is used for large and high performance applications which stores data.

### Steps for Installation.

#### Step 1: Downloading MYSQL.
1. Go to your web browser and search https://www.mysql.com/downloads/
2. Click on download now and download according to your laptop operating system.

#### Step 2: Intallation of MYSQL.

1. Open the downloded file.
2. Once opened choose "my server only".
3. Set the port number to default.
4. Create a root password then finish the configuration.

## Postgre.

### Step 1: Download PostgreSql.

1. Go to your browser https://www.postgresql.org/download/
2. Choose your Operating system.
3. After downloading open it then choose componencts for installation then click on next to continue.

### Step 2: Configuration of Postgresql.
1. One will be asked to configure some essential settings which are:
- You will be required to set a password.
- You will be asked to input aport number leave it at default which is '5432'.
2. Leave the installation at locale default settings then finish installation.

### What about for other users that is Macbook and Linux.
 One can use a Virtual machine to access this tools.

 ### Step 1: Virtualbox.
 1. Install VirtualBox and select for Macbook or Linux.
 2. Follow the installation instructions and the virtualbox will be fully installed into your machine.
 3. Download windows ISO file and save in you laptop.

### Step 2: Create a virtual machine.
1. Open the virtualbox and then click on new.
2. You will be required to fill;
- Name of the virtual machine.
- The Type.
- Version of the windows.

### Step 3: Allocate virtual machine some memory.
1. Choose how much RAM to give the virtual machine.

### Step 4: Create the hard disk of the VM.
1. Click on create a virtual hard disk now then click next.
2. Choose Virtualbox Disk image.
3. Choose the dynamically allocated then set size to 50GB.

### Step 5: Install windows.

1. Select the virtualmachine you created then click on start.
2. You shall be prompted then select the windows ISO file that we downloaded and folow the normal windows installation steps.
3.The windows will open inside the virtual machine.

### Step 6: Installation Excell and PowerBI.

1. Inside the windowss viryual machine go to Microssoft Store and download both PowerBI and excel.

### Step 7:Launch the application.

1. Inside the Windows virtual machine you can now open;
- PoweBI.
- Excel.
